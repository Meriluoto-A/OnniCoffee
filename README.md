☕ **OnniCoffee**

OnniCoffee is a fictional Nordic coffee roastery and café chain used as an analytics case.
The focus is on understanding how price, weather, inflation and customer behaviour affect demand and profitability.

❓ Core Question
+ How do price, weather, inflation and customer behaviour influence demand and optimal pricing?

📁 Data

+ Café‑level transaction data (time, location, basket)
+ Coffee price history and inflation corrected price
+ Effect of weather to coffee price (temperature, rainfall)

🔍 Key Analyses

+ Sales pattern clustering (time of day, weekday vs weekend)
+ Correlation analysis: weather, inflation and demand
+ Pricing scenario (€3 → €4, demand impact)
+ Delivery route optimisation

🧠 Methods

+ pandas for data preparation
+ KMeans clustering
+ Regression models (price → demand)
+ Basic optimisation (price × volume × margin)

📊 Outputs

+ Python: demand curves, clusters, scenario charts
+ Power BI: café and market performance dashboards


✅ Skills Demonstrated
Econometric thinking · Pricing analysis · Clustering · Regression · Forecasting · Optimisation
